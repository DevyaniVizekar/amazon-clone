# amazon-clone
A responsive Amazon homepage clone built using HTML and CSS.  
This project focuses on replicating the layout and design of a real-world e-commerce website.

## Features
- Amazon-like homepage layout
- Responsive design
- Navigation bar and product sections
- Clean and structured UI

## Tech Stack
- HTML
- CSS

## What I Learned
- Building complex layouts using CSS
- Working with Flexbox and positioning
- Structuring large web pages
- Improving UI design skills

## Project Status
This is a frontend-only project and does not include backend functionality.
